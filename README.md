# Introduction
React and gp web

## How to start
```bash
npm install
npm start
```
Navigate to https://localhost:3000

## How to deploy
```bash
npm run build
npm install --save-dev gh-pages
npm run deploy -- -m "Deploy React app to GitHub Pages"
```
